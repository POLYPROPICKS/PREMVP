## 1. Root cause (reservation layer — deterministic)

`lib/executor/nightEventReservations.ts` `buildReservationPlan`:
former line ~370 hard-gated every physical-match group on
`eventTierOf(best) === "TIER1"`, else `skippedNonTier1 += 1; continue`.

Consequence: any real physical match whose **best allowed full-match anchor is Tier2 or
Tier3** (score/coverage below the 72/50 Tier1 bar) was **silently dropped** — no
reservation, no slot, invisible. This is exactly the founder-rejected
`TRUE_MISSING_RESERVATION_HAS_ALLOWED_FULLMATCH` shape: an allowed full-match market
exists but no reservation is produced.

Classification vs 78b192f: **PREVIOUS_FIX_INCOMPLETE / NEW_EDGE_CASE_SAME_BUG_FAMILY**.
78b192f fixed weak-single-team-key *merge* underfill (keys collapsing wrongly), but left
the Tier1-only event gate intact, so the sub-Tier1 full-match case still underfilled.

The founder live-slot policy (Tier1 → Tier2 → Tier3 fallback to fill ≥15 slots, never
forbidden classes) was **not implemented in the reservation layer at all** — only a
Tier2 *live-execution* override existed inside the builder, which does not create
reservations.

## 2. Patch (one file)

`lib/executor/nightEventReservations.ts`:
- Added `isAllowedFullMatchAnchor()` — positive full-match matcher
  (moneyline / match winner / spread / handicap / total goals O/U), corners and all
  forbidden classes excluded via existing `isForbiddenAnchorMarket`.
- Added `TARGET_LIVE_SLOTS = 15`.
- Reworked group loop: Tier1 groups reserved as before; sub-Tier1 groups with a
  **positively-allowed full-match anchor** are held in `fallbackRankable`
  (tagged TIER2/TIER3).
- Slot-fill ladder: after Tier1, promote fallback groups **Tier2 before Tier3**, only up
  to `TARGET_LIVE_SLOTS` (never beyond). Each is marked `FALLBACK_SLOT_FILL` in
  `selection_reason` and `diagnostics.slot_fill` / `diagnostics.fallback_tier`.
- Groups with NO allowed full-match anchor → counted in
  `fallbackSkippedNoAllowedFullmatch` (forensic visibility, not silent).
- New diagnostics: `targetLiveSlots`, `tier1ReservedCount`,
  `fallbackSlotFillReservedCount`, `fallbackTier2Reserved`, `fallbackTier3Reserved`,
  `fallbackEligibleGroupsSeen`, `fallbackSkippedNoAllowedFullmatch`,
  `slotFillTargetReached`.

Invariants preserved: forbidden anchors never selected; representative title never a
forbidden market; horizon gate unchanged; stake/cap untouched (reservation layer sets no
stake — stake stays in the builder); idempotent persist unchanged.
